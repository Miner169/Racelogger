🔑 Admin Zone — New Event Setup

Archives the entire Google Sheet (Racelog + Safety Notes) to a timestamped backup tab, then clears it for a brand-new event — the same result as opening the spreadsheet and doing it by hand, without leaving the app. Every connected device also gets its local log store cleared automatically the next time it syncs. Requires the admin token (set as ADMIN_TOKEN in the Apps Script's Script Properties) — volunteers without it cannot run this.

📊 Race Metric Parameter Index
  • 🔹 Categories: Which distance category this runner belongs to.
  • 🔹 Lap/CP & Time: The specific checkpoint lap marker index.
  • 🔹 Time per lap/CP: Split timeline elapsed since prior checkpoint.
  • 🔹 Total time: Overall accumulated timeline duration since start.
  • 🔹 Pace: Time taken per distance (min/km). (Lower = faster pace)
  • 🔹 Speed: Calculated rate of movement (km/h). (Higher = faster speed)
  • 🔹 Projected Finish: Expected total completion time based on current performance.
